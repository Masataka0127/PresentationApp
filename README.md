# PresentationApp
録音機能を持ったタイマーのiOS/Androidアプリです。

# Install  
- ```git clone 'https://github.com/Masataka0127/PresentationApp.git'```  

- ```npm install```  

## versions  
- Node: v10.6.0  
- npm: 6.1.0
- yarn: 1.7.0 
- git: 2.18.0
